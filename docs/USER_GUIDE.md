# Magic Imger — быстрый старт (пользователь)

Цель этого файла — максимально простой запуск.

## 1) Установи Node.js

Нужен Node.js `>= 22`.

## 2) Запуск на Windows (в один клик)

Запусти `run-win.vbs` (без окна терминала).

Если нужно видеть лог/ошибки в консоли — запусти `run-win.bat`.

Он:

- проверит Node.js;
- автоматически поставит npm-зависимости (если `node_modules/` отсутствует);
- запустит GUI.

## 3) Запуск на macOS

Запусти `run-mac.command`.

## 4) Запуск на Linux/macOS из терминала

```bash
chmod +x ./run-unix.sh
./run-unix.sh
```

Если скрипт запущен “двойным кликом” из файлового менеджера, он попробует сам открыть окно терминала, чтобы было видно лог.

## CLI (опционально)

```bash
./run-unix.sh cli -- --help
```

Примеры команд и параметры — в `README.md`.

## GUI: что куда вводить (коротко)

### Images

1) Добавь файлы (drag & drop в панель Queue или **Add Files**)  
2) Укажи **Output folder** (обязательно)  
3) Выбери **Output format** (если нужно)  
4) Нажми **Process Queue** (или **Plan Batch** для оценки RAM/времени)

### Screenshots

1) Вставь URL (по одному в строке)  
2) Укажи **Output folder** (обязательно)  
3) Выбери **Browser mode** (по умолчанию подходит `bundled chromium`)  
4) Нажми **Run**
