# Magic Imger

Программа для нескольких простых задач с изображениями,

1) Сжатие изображений
2) Ресайз изображений \ смена формата
3) Массовая нарезка скриншотов
(＾▽＾)／

## Самое главное: как запустить

Скрипт запуска **сам подхватит Node.js**:
- если Node.js не установлен / не той разрядности / сломан — он скачает портативный Node.js в папку `.runtime` и запустит всё сам (без админа).
- если хочется «починить всё» — есть режим `repair` (см. ниже).

Дальше просто запускай файл:

- **Windows:** `run-win.vbs` (без чёрного окна)  
  Если нужно увидеть ошибки — `run-win.bat`.
- **macOS:** `run-mac.command`
- **Linux/macOS (если запускаешь из терминала):** `./run-unix.sh`

Скрипт сам поставит нужные пакеты (если их ещё нет).

## Как пользоваться (очень коротко)

### Images (картинки)

1) Перетащи картинки в окно (или нажми **Add Files**)  
2) Укажи **Output folder** (обязательно; куда сохранить результат)  
3) Нажми **Process Queue**

Результат будет в папке **Output folder** (она в приложении кликабельная).

(ง'̀-'́)ง

### Screenshots (скриншоты по URL)

1) Переключись на **Screenshots**  
2) Вставь ссылки (по одной в строке)  
3) Укажи **Output folder** (обязательно)  
4) Нажми **Run Screenshot Batch**

Результат будет в папке результата справа (кликабельная ссылка).

## Если не запускается

- На Windows запусти `run-win.bat`, чтобы увидеть текст ошибки
- Попробуй «само-починку»: `repair-win.bat` (или `run-win.bat repair`)
